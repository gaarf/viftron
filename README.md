VifTron
==========

_an OnVif client built with Electron_

### Installation

Global dependencies:

* `npm install -g bower gulp`

Local dependencies:

* `npm install && bower install`

### Running the server

* `npm start`

### Building app files into dist/

* `gulp build`
or
* `gulp watch` (autobuild)

### All together now

* `gulp develop` (autobuild, autorestart)
